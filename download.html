4. Combined Percentage of Households to Total Households by Amenities and Assets (Maharashtra - Taluka, Village and Ward Level)


From the above Marged Excel File, if I need a specific district data, I just need to put a filter in the DT Name ( District Name) column.
District Code (filter any District Code) = That district all details of Households Amenities and Assets,
District Name (filter any District Name) = That district all details of Households Amenities and Assets,
Tehsil Code or (State Code/Name + Tehsil Code) = Any Talukas all details, Tehsil Code (00000)= Every District Total, Rural, Urban All details. Then from the Rural/Urban column, we can select all Maharastra all district-wise total (Total, Rural, Urban) Tehsil Code (99999)= Area not Under any Sub-district(count 103 in Maharastra)
Ward No (00000)= Every District Total, Rural, Urban All details. Then from the Rural/Urban column, we can select all Maharastra all district wise total (Total, Rural, Urban)

Ward No + Area Name (Select District) + (Total, Rural or Urban) = District level all details
Ward No + Area Name (Select Sub-District) + (Total, Rural or Urban) = Sub-District level all details
Ward No + Area Name (UnSelect District and Sub-District) = Rural Level All details
Key Attributes: "Town Code/Village code"
